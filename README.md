# Github Notes

## Git

A Version control software that keeps track the changes in your code over time.

## Github

A Website that you can use <strong>Git</strong> on.

## Repo

A Folder which contains your files.

## Commit

Saves what you wrote in a file.

## Clone

Make a copy of entire repo from remote to local.

## Push

Send commits from local to remote.

## Pull

Receive commits from remote to local.

# Commands

## PWD

Print Working Directory
Tells you where you are

## ls

List

## cd

Change directory
Navigates to that directory

## mkdir

Make directory (folder)

## rmdir

Remove directory (folder)

## .. 

Represents the parent directory
